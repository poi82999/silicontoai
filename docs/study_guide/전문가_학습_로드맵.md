### 교재

#### 1. Computer Organization and Design: RISC-V Edition — Patterson & Hennessy (5판)
- **출판사**: Morgan Kaufmann (2020)  
- **ISBN**: 978-0128203316  
- **읽을 챕터**: 1, 4(파이프라인 전체), 5(메모리 계층), 6(병렬 입문)  
- **이 프로젝트 연결**: 4장 파이프라인 해저드 → systolic array의 skew/deskew 15사이클이 왜 필요한지; 5장 캐시 → SRAM 뱅크 설계 근거  
- **구입**: 교보문고 / Amazon / O'Reilly 구독  

#### 2. Computer Architecture: A Quantitative Approach — Hennessy & Patterson (6판)
- **출판사**: Morgan Kaufmann (2019)  
- **ISBN**: 978-0128119051  
- **읽을 챕터**: 1(성능 측정), 2(메모리 계층 심화), 4(데이터레벨 병렬성/SIMD), Appendix C(파이프라인 심화)  
- **이 프로젝트 연결**: 4장 SIMD → 16-lane MAC의 설계 근거; Roofline 모델 → `l6/src/l6_toolchain/roofline.py`의 이론 배경  
- **참고**: 1권 먼저 읽고 여기서 심화 챕터만 골라 읽으면 됨

## "대체 불가" 포지셔닝 전략

AI가 코드를 쓰는 시대에 살아남는 방법은 **"빠르게 쓰는 것"이 아니라 "왜 그 설계인가를 판단하는 것"** 입니다.

### 차별화되는 3가지 조합

```
조합 1: RTL + 컴파일러 동시 이해
  → "이 conv2d를 SA에서 효율적으로 실행하려면
     tiling 전략을 어떻게 바꿔야 하는가?"
  → 두 도메인을 모두 아는 사람만 답 가능

조합 2: 알고리즘 + 하드웨어 공동 최적화
  → FlashAttention처럼 알고리즘 자체를
     하드웨어 메모리 계층에 맞게 재설계
  → 논문 저자 수준의 인사이트

조합 3: 시뮬레이션 + 실 칩 검증 경험
  → Verilator 시뮬 → FPGA → ASIC으로 이어지는
     검증 체인을 직접 밟아본 경험
  → 인턴십/취업 면접에서 즉시 차별화
```

### 3년 후 이력서 항목

```
- 16×16 Weight-Stationary Systolic Array NPU 풀스택 구현 (RTL → 컴파일러)

- UVM functional coverage 90%+ 검증 환경 구축
- FPGA bringup + PYNQ-Z2 ARM 제어 완료
- TVM/MLIR 오픈소스 기여 (PR 링크)
- [선택] ASIC 테이프아웃 경험 (OpenMPW)
```
